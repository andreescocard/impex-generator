## 📖 Overview

The **IMPEX Generator** is a tool that generates sample IMPEX scripts for **SAP Commerce (Hybris)**. It streamlines the creation of common IMPEX configurations, so configuring and maintaining your SAP Commerce environment takes minutes instead of hours.

## ✨ Features

Form-driven generators for the most common SAP Commerce IMPEX scripts:

- 🚩 **Feature Flag**
- 📝 **Paragraph**
- 📄 **Page**
- 🖼️ **Product Images**
- 📦 **Product**
- 🗂️ **Category**
- 🔗 **CMS Link Component**
- 🧩 **Content Slot**
- 💲 **Price Row**

Plus:

- 💾 **Snippet library** — save and reuse your own configurations (localStorage)
- 🧱 Clean UI built with [Spartan-ng](https://www.spartan.ng/) + Tailwind
- ⚡ Powered by Angular + Analog inside an Nx monorepo

## 🚀 Getting Started

```bash
# install dependencies
npm install

# start the dev server
npx nx serve impexgenerator
```

Then open the app in your browser, pick a generator, fill the form, and copy the generated IMPEX script.
